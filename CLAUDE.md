## AI Labor Forecast — Scoring Pipeline

The workbook is `jobs-data-v3.xlsx`. The methodology PDF is generated by `docs/generate_methodology.py`.

### Master Equation
`d(t; i, s) = d_max(i) * phi(a(i,s)) * E(i) * T(t; i, s) * R(i)`

### Three Layers
1. **Task-level:** 3,552 tasks × 2 scenarios = 7,104 autonomy fraction scores (already scored, in `3 Tasks` sheet)
2. **Job-level:** 462 occupations. `a = x_depth * avg(x_scale, x_sub)` where `x_depth = task_coverage * workflow_simplicity`
3. **Industry-level:** 17 sectors. d_max, E, T, R applied from frictions tabs.

### Job-Level Scoring Pipeline (workflow_simplicity, x_scale, x_sub)
These 3 variables are scenario-independent (structural job properties). 462 × 3 = 1,386 scores.

- **Phase 0 — Pre-processing:** Extract structured profiles from workbook → `scoring/job_profiles.json`. Includes task stats, GWA distributions, construct-specific features (interpersonal/digital/judgment shares).
- **Phase 1 — Calibration (1 agent, foreground):** Score 30 boundary-case occupations with full reasoning. Output = anchor exemplars. Must complete before Phase 2.
- **Phase 2 — Batch Scoring (6 parallel agents):** Split 462 jobs by sector groupings. Each agent gets anchors + rubrics + profiles. Joint scoring with independent reasoning blocks per variable. Output = JSON per agent.
  - Agent A: Healthcare & Life Sciences (116)
  - Agent B: Education & Academia (74)
  - Agent C: Government & Public Admin (59)
  - Agent D: Finance, Insurance, Accounting, Law, Consulting, Real Estate (69)
  - Agent E: Technology, Architecture & Engineering (67)
  - Agent F: Advertising, Media, Manufacturing, Retail, Logistics, Energy (77)
- **Phase 3 — Auditor (1 agent):** Programmatic checks (distribution skew, correlation caps, SOC-group coherence) + LLM re-scoring of flagged ~10-15%.
- **Phase 4 — Reliability (1 agent):** Resample 46 jobs (10%), re-score, compute weighted quadratic kappa. Thresholds: κ ≥ 0.75, within-one-step ≥ 90%, exact ≥ 65%.
- **Phase 5 — Write-back:** Write final scores to `4 Results` tab columns: workflow_simplicity, x_scale, x_sub.

### Scoring Scales
- **workflow_simplicity:** 5-point {1.00 Independent, 0.75 Mostly independent, 0.50 Moderate interdep., 0.25 Tightly coupled, 0.00 Fully dynamic}
- **x_scale:** 5-point {0.00 No throughput gain, 0.25 Modest, 0.50 Moderate, 0.75 Strong, 1.00 Near-unlimited}
- **x_sub:** 5-point {0.00 Human IS the product, 0.25 Strong human pref, 0.50 Mixed, 0.75 Weak human pref, 1.00 Fully substitutable}

### Key Design Rules
- x_sub = human IS the product (intrinsic); T3 = customers not yet willing (preference, eroding)
- Joint scoring with forced independent reasoning blocks to mitigate halo effects
- Task-level features curated per variable to avoid construct contamination (x_sub gets interpersonal/judgment shares; x_scale gets digital/physical shares)
